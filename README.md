# Current version: Demo v0.6 — Country → City navigation + Search bar

CoGrazer VR is a concept project by **Ahmet**, a future tech entrepreneur.  
The goal is to let people explore real cities remotely using VR headsets and physical robots.

This v0 demo shows a simple city switcher using embedded Google Maps views.

## Features (v0)

- Cyber-tech dark UI (neon cyan + black)
- City selector: Gökçeada, Paris, London
- Embedded map viewer that switches based on the selected city
- Placeholder movement controls (for future robot control)

## Roadmap

- **v1:** VR-friendly layout & smooth controls  
- **v2:** Connect to real mobile robots streaming live video  
- **v3:** Multi-city robot network with subscriptions and partnerships
